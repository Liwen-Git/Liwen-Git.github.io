遍历二叉树

深度优先遍历 DFS

二叉树的深度优先遍历可以细分为：先序遍历、中序遍历、后续遍历。

  • 前序遍历：根节点 -> 左子树 -> 右子树
  • 中序遍历：左子树 -> 根节点 -> 右子树
  • 后续遍历：左子树 -> 右子树 -> 根节点
广度优先遍历 BFS

广度优先遍历：又叫层次遍历，从上往下对每一层依次访问，在每一层中，从左往右（也可以从右往左）访问结点，访问完一层就进入下一层，直到没有结点可以访问为止。

判断是否是二叉搜索树

判断一颗树是否是二叉搜索树，一棵树是BST需要满足

  • 一个节点的值大于它左子树所有节点的值
  • 一个节点的值小于它右子树所有节点的值
  • 左右子树也必须是二叉搜索树

所以只需要遍历每个节点，判断

  • 该节点值是否大于左子树的最大值
  • 该节点值是否小于右子树的最小值
